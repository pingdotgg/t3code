## Appendix: Raw Audit Notes

### Scope

- **Searched:** All app/packages source code, `docs/`, `.github/` workflows, `.gitignore`, `.vscode/`, `tsconfig.base.json`, `.env.example`, all `package.json` files
- **NOT searched:** `.repos/` (vendored, out of scope), `experiments/`, `.plans/`, `patches/`, `assets/`, `.cursor/`, `.macroscope/`

### Methodology

Audit performed by 5 parallel subagents:

1. **Docs & README** — scanned all documentation for typos, broken links, stale content, missing READMEs
2. **Missing Tests** — identified source files without test files, TODO tests, test gaps
3. **Simple Bugs & Lint** — searched for `@ts-ignore`, `as any`, empty catches, bare `console.error`, swallowed errors
4. **Code Quality & Dead Code** — identified unused exports, magic numbers, missing return types, dead code
5. **Config & CI** — audited CI workflows, gitignore, TypeScript config, VS Code extensions, env vars, build scripts
